# Objetivo Oficial do Programa

```txt
Carnegie Mellon's online graduate certificate focuses on deep, technical understanding of
generative AI systems. The program emphasizes theory, data representation and systems
scalability across three core courses covering large language models, multimodal machine
learning, and LLM systems architecture.

Program participants will develop foundational skills for understanding, working with, and
adapting existing tools and technologies; explore the fundamental mathematical concepts in
machine learning and deep learning; and learn how to design and implement scalable LLM
systems.
```

**Período publicado:** Fall 2026 (no certificado); Fall 2025 (edição pública análoga 11-667)
**Código oficial:** 11-967 (certificado online) / 11-667 (curso regular presencial)

### 📖 Disciplina: Large Language Models: Methods and Applications

- **Carga Horária:** Não informado publicamente
- **Professor(es):** Daphne Ippolito, Fernando Diaz (edições anteriores); Jaromir Savelka, Yubin Kim (Fall 2025 — 11-667); lista completa do certificado inclui todos os 7 professores LTI listados

#### 📝 Ementa

> Fonte: https://www.cmu.edu/online/gai-llm/curriculum/index.html + https://2025.cmu-llms.org/syllabus/ (acesso 2026-05-15)

```txt
Foundational course covering understanding, working with, and adapting LLMs like BERT, T5,
and GPT. The course provides a holistic view of the current state of large language models.
The first half covers fundamentals including network architectures, training, inference,
evaluation, interpretation, alignment, and emergent capabilities. The second half addresses
scaling techniques, efficient pretraining and deployment, deployment concerns, and frontier
challenges.

Topics include:
- Transformer architectures and comparison to LSTM models
- Data filtering, training objectives, RLHF/instruction tuning
- Evaluation methodologies and ethical considerations
- Decoding strategies and pretraining techniques
- PyTorch implementation of neural language models
- Fine-tuning with open-source libraries (HuggingFace)
- Downstream applications and novel methodologies
```

#### 📝 Ementa

> Fonte: https://www.cmu.edu/online/gai-llm/curriculum/index.html + https://llmsystem.github.io/llmsystem2025spring/ (acesso 2026-05-15)

```txt
Systems-focused course on designing and implementing scalable LLM infrastructure.
The course covers how training, serving, fine-tuning, and evaluating LLMs require
sophisticated engineering with modern hardware and software stacks.

Topics include:
- GPU programming basics and GPU acceleration for transformers
- Learning algorithms and automatic differentiation
- Deep Learning frameworks design
- Transformer architecture at the systems level
- Tokenization techniques
- LLM decoding strategies
- Distributed model training (GPipe, DeepSpeed, ZeRO, communication-efficient algorithms)
- Efficient fine-tuning: PEFT methods, LoRA, data-efficient fine-tuning
- Model quantization (Parts I–II)
- Mixture-of-Expert systems
- Attention optimization (FlashAttention)
- LLM serving: vLLM/PagedAttention, KV cache optimization, disaggregated prefill-decoding
- Structured generation language (SGLang)
- Reinforcement learning systems (RLHF implementation)
- Optional advanced topics: RAG systems, multimodal LLMs, speculative decoding, JIT compilation
```

#### 📝 Ementa

> Fonte: https://www.cmu.edu/online/gai-llm/curriculum/index.html + https://cmu-mmml.github.io/ (acesso 2026-05-15)

```txt
Mathematical foundations for integrating multiple data modalities in machine learning
systems. Addresses integrating and modeling multiple communicative modalities, including
linguistic, acoustic, and visual messages.

Six main challenges in multimodal machine learning:
1. Representation — multimodal representation learning
2. Alignment — modality alignment techniques
3. Reasoning — multimodal reasoning approaches
4. Generation — translation and mapping between modalities
5. Transference — co-learning methodologies
6. Quantification — multimodal auto-encoders, deep canonical correlation analysis

Additional topics:
- Multimodal transformers
- Multimodal tensor fusion
- Mutual information in multimodal systems
- Multi-kernel learning and attention models
- Multimodal graph networks
- Applications: affect recognition, image/video captioning, cross-modal retrieval,
  language grounding, language-vision navigation
```

# Cursos-Irmãos Top-Tier Identificados

## Stanford CS324 — Large Language Models

- **Instituição:** Stanford University
- **URL do material público:** https://stanford-cs324.github.io/winter2022/
- **Por que é curso-irmão:** Mesma temática central (LLMs), nível graduate, cobre arquitetura, treinamento, alignment e deployment; professores Percy Liang e Tatsunori Hashimoto frequentemente citados em papers seminais usados no CMU 11-667.

## Princeton COS597G — Understanding Large Language Models

- **Instituição:** Princeton University
- **URL do material público:** https://www.cs.princeton.edu/courses/archive/fall22/cos597G/
- **Por que é curso-irmão:** Foco em fundamentos teóricos de LLMs (scaling laws, emergent abilities, in-context learning), sobreposição direta com semanas 1–9 do CMU 11-667.

## Berkeley CS294-267 — Understanding Large Language Models

- **Instituição:** UC Berkeley
- **URL do material público:** https://rdi.berkeley.edu/understanding_llms/s24
- **Por que é curso-irmão:** Cobre o mesmo espectro técnico (pretraining, RLHF, RAG, agents, safety), com material público de 2024; professores frequentemente co-autores de papers listados no CMU 11-667.

## MIT 6.S191 — Introduction to Deep Learning

- **Instituição:** Massachusetts Institute of Technology
- **URL do material público:** http://introtodeeplearning.com/
- **Por que é curso-irmão:** Fundamentos de deep learning que sustentam os três cursos do certificado; material de referência para gap supplementation em pré-requisitos de arquitetura e treinamento.

```txt
COURSE 1: Large Language Models: Methods and Applications (11-967)
Fall 2026 | Tuition: $8,484

Foundational course covering understanding, working with, and adapting LLMs like BERT,
T5, and GPT.

Topics Include:
- Transformer architectures and comparison to LSTM models
- Data filtering, training objectives, RLHF/instruction tuning
- Evaluation methodologies and ethical considerations
- Decoding strategies and pretraining techniques
- PyTorch implementation of neural language models
- Fine-tuning with open-source libraries
- Downstream applications and novel methodologies

Additional Costs: AWS ($150–$300) and OpenAI API (up to $25) required for assignments

COURSE 2: Large Language Model Systems (11-968)
Spring 2027 | Tuition: $8,484

Systems-focused course on designing and implementing scalable LLM infrastructure.

Topics Include:
- Training, serving, fine-tuning, and evaluation approaches
- Efficient embedding storage and retrieval
- Data-efficient fine-tuning techniques
- Communication-efficient algorithms
- GPU acceleration and model compression
- Reinforcement learning with human feedback implementation
- Online maintenance and latest advances

Additional Costs: AWS and/or OpenAI services (up to $300)

Fall 2027 | Tuition: $8,484

Mathematical foundations for integrating multiple data modalities in ML systems.

Core Concepts:
- Multimodal representation learning
- Translation and mapping between modalities
- Modality alignment techniques
- Multimodal fusion approaches
- Co-learning methodologies
- Multimodal auto-encoders and deep canonical correlation analysis
- Multi-kernel learning and attention models
- Applications: affect recognition, image/video captioning, cross-modal retrieval

Additional Costs: AWS and/or OpenAI services (up to $300)
```
