# MBA em Engenharia de Dados, IA e Arquitetura de Plataformas

## Duração

12 meses

## Carga Horária

Aproximadamente 800 horas

## Modalidade

Programa técnico profissional intensivo baseado em:

- estudo dirigido
- desenvolvimento de projetos reais
- laboratórios técnicos
- preparação para entrevistas internacionais
- certificações profissionais

## 1. Visão Geral do Programa

O MBA em Engenharia de Dados, IA e Arquitetura de Plataformas tem como objetivo formar profissionais capazes de projetar, desenvolver e operar plataformas modernas de dados, inteligência artificial e inteligência generativa em escala empresarial.

O programa combina:

- Engenharia de Dados
- Machine Learning Engineering
- MLOps
- Sistemas Distribuídos
- Arquitetura de Cloud
- IA Generativa
- Arquitetura de Plataformas de IA

Ao longo do programa, o aluno evolui progressivamente de Machine Learning Engineer para AI Platform Architect, desenvolvendo habilidades técnicas profundas e também competências necessárias para atuar em empresas globais de tecnologia e consultorias internacionais.

## 2. Objetivo Geral

Formar profissionais capazes de projetar, implementar e operar plataformas modernas de dados e inteligência artificial em ambientes distribuídos e cloud-native, habilitando-os a atuar em empresas globais de tecnologia e em projetos de grande escala.

## 3. Objetivos Específicos

Ao final do programa, o aluno será capaz de:

### Engenharia de Dados

- Construir pipelines de dados escaláveis
- Implementar arquiteturas Lakehouse
- Trabalhar com processamento batch e streaming
- Projetar pipelines de dados resilientes

### Machine Learning Engineering

- Desenvolver modelos de Machine Learning clássicos e Deep Learning
- Implantar modelos em produção
- Construir APIs de inferência
- Implementar pipelines de treinamento automatizado

### MLOps

- Implementar pipelines CI/CD para ML
- Monitorar modelos em produção
- Detectar drift de dados e conceito
- Automatizar retraining

### Arquitetura de Cloud

- Projetar sistemas distribuídos
- Utilizar AWS, Azure e GCP
- Implementar soluções serverless e containerizadas

### GenAI Engineering

- Construir sistemas RAG
- Desenvolver agentes baseados em LLMs
- Avaliar e monitorar aplicações de IA generativa
- Implementar guardrails de segurança

### Arquitetura de Plataformas

- Projetar plataformas de dados corporativas
- Criar arquiteturas multi-tenant
- Implementar observabilidade e governança
- Definir estratégias de escalabilidade

### Preparação para Big Tech

- Resolver problemas de algoritmos
- Projetar sistemas distribuídos
- Participar de entrevistas técnicas
- Construir portfólio técnico sólido
